## Core Concept

[**Machine learning**](https://www.ibm.com/think/topics/machine-learning) is a subset of Artificial Intelligence (AI) that focuses on building algorithms capable of **learning from data**.

Instead of relying on hard-coded rules, machine learning models:
- Analyze historical (training) data  
- Identify meaningful patterns  
- Use those patterns to make predictions or decisions on new, unseen data  

### Why Machine Learning Matters

Machine learning powers many real-world systems, including:
- Search engines and recommendation systems  
- Fraud detection and risk analysis  
- Speech recognition and language translation  
- Image and video understanding  

Its ability to scale with growing data makes ML a core technology in todayâ€™s data-driven world.
